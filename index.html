Trabalhando com versionamento

#1 - Configuração da Máquina

Essa configuração é realizada globalmente para setar as informações na máquina tais como o Nome e E-mail do desenvolvedor.

Definindo nome de usuário: git config --global user.name "Rubens Araújo"
Definindo e-mail do usuário: git config --global user.email "rubens@octosoft.com.br"
Definindo Editor do GIT: git config --global core.editor "code --wait"
#2 - Comandos do GIT
git --version - Exibe a versão instalada em nossa máquina.
git config --list - Exibe a lista de todas as configurações globais realizadas.
git config user.nome - Exibe o nome do usuário git configurado em nosso PC.
git init - Rode esse comando dentro da pasta do seu projeto para inicialziar um repositório local.
git status - Exibe o status do nosso repositório.
git add -A - Adiciona no mapemaneto apenas os arquivos que teve alteração.
git add * - Adiciona no mapemaneto todos os arquivos que teve alteração.
git commit -m "Sua Mensagem" - Cria uma versão de nosso código com a mensagem.
git log - Exibe todos os commits realizados em nosso repositório
git branch - Exibe em qual branch estamos trabalhando.
git reset --soft - Reseta os arquivos na posição atual, já com os arquivos adicionados ao reposiório.
git reset --mixed - Reseta os arquivos na posição atual sem o commit e sem estar com os arquivos adicionados.
git reset --hard - Reseta e volta para o commit selecioniado, todos os arquivos do commit mais recente será apagdo.
#3 - Criando Branchs
git branch teste - Com isso criamos uma branch nova chamada teste.
git checkout teste - Altera nossa branch
#4 - Exibindo alterações dos arquivos
git diff - Exibe as alterações que tiveram nos arquivos.
git diff --name-only - Exibe apenas o nome dos arquivos que foram alterados.
git diff index.html - Exibe as aterações que houve no arquivo.
git checkout HEAD -- README.md - Apaga as alterações que foram realizadas.
#5 - Enviando nosso repositório local para o GIT
ssh-keygen -t ed25519 -C "your_email@example.com" - Será gerada uma chave pública e privada na pasta de nosso usuário, copiei a chave pública e adicione nas configurações do seu perfil do git em Settings -> SSH and GPG Keys.
Criei um repositório em sua conta do GIT.
git remote add origin https://github.com/RubensAraujoSilva/modulogit.git - User o comando gerado no repositório
